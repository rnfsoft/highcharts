2. superuser ID: admin, Password:DjangoPassword
3. Run batch files to invoke 
	_initialize: Download basic packages, Python and Django, first time only to start a project
	_migration: Migrating model changes
	_requirements_export: Exporting requirements to requirements.txt
	_requirements_import: Importing packages listed in requirements.txt
	_runserver: local for development, production for live
	_startapp: Creating a new app 	
	_virtualenv: Activating virtual enviroment Shell
